# 🤖 Ин-Кон Суперагент (DeepSeek-V3 / R1)

Высокотехнологичный и проработанный интерфейс чата с полноценным **Режимом Суперагента (Tool Use)** на базе моделей DeepSeek. 

Приложение спроектировано для развертывания на вашем VPS-сервере в один клик.

## ✨ Ключевые возможности

1. **Режим Суперагента (V3)**: При отключенной опции *DeepThink (R1)*, модель работает в режиме автоматического выполнения системных инструментов на вашем VPS:
   * 📟 **Shell Терминал (`run_command`)**: Выполнение bash/sh команд и возврат stdout/stderr (например, проверить ресурсы, перезагрузить docker, запустить скрипты).
   * 📂 **Файловый менеджер (`read_file` / `write_file`)**: Чтение, создание, редактирование и перезапись любых конфигураций или файлов баз данных прямо на сервере.
   * 🔍 **Поиск по веб сети (`web_search`)**: Нативный парсинг результатов поиска в реальном времени.
   * 📄 **Импорт документов (`scrape_url`)**: Быстрый сбор ссылок и разбор статей/веб-ресурсов в чистый текст.
2. **Премиальный режим DeepThink (R1)**: Глубокое размышление с отображением детального лога мыслей и интерактивным временем вычислений в реальном времени.
3. **Настоящая Автосинхронизация сессий**: Все диалоги автоматически сохраняются в формате JSON в директории `data/conversations/` на сервере. Ваши данные никогда не пропадут при очистке кэша браузера и будут доступны с любых устройств.
4. **Ультимативный дизайн**: Полностью русскоязычный интерфейс, интерактивный просмотр вызовов API и параметров инструментов в стиле терминала, адаптивная клавиатурная навигация.

## 🚀 Быстрый запуск на VPS через Docker (Рекомендуемый способ)

Для запуска нужен только установленный Docker и Docker Compose.

1. Перенесите файлы проекта (`Dockerfile`, `docker-compose.yml`, `package.json`, `server.ts` и т.д.) на ваш сервер VPS.
2. Откройте файл `docker-compose.yml` и замените `your_deepseek_api_key_here` на ваш реальный ключ API DeepSeek:
   ```yaml
   environment:
     - NODE_ENV=production
     - DEEPSEEK_API_KEY=sk-xxxx... # Ваш API-ключ
   ```
3. Запустите суперагента командой:
   ```bash
   docker compose up -d --build
   ```
4. Приложение будет доступно по адресу `http://<IP_Вашего_Сервера>:3000`.

*Все диалоги будут автоматически и безопасно сохраняться в папку `./data` на вашем хост-компьютере VPS.*

## 🛠 Ручной запуск (без Docker)

Если вы предпочитаете менеджеры процессов NodeJS (например `pm2`):

1. Установите NodeJS (v18 или v20+) на VPS:
   ```bash
   npm install
   ```
2. Создайте файл `.env` в корневой папке проекта:
   ```env
   DEEPSEEK_API_KEY=sk-your-deepseek-key-here
   PORT=3000
   ```
3. Соберите и запустите проект:
   ```bash
   npm run build
   # Запуск через PM2 для фоновой работы:
   pm2 start dist/server.cjs --name "incon-superagent"
   ```

## 📦 Структура хранения данных
* `/data/conversations/` — хранилище всех сессий в читаемом формате JSON. При перезапусках контейнера или обновлении кода вся история диалогов надёжно сохраняется.
